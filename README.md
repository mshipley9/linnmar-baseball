# Linn-Mar Baseball Stats

Historical baseball statistics platform for Linn-Mar Baseball, Marion, Iowa (1981–Present).

## Pages

- **linn-mar-hitting-stats.html** — Standalone hitting statistics platform with career stats, season stats, records, and player comparison
- **linn-mar-baseball-stats.html** — Full program platform with team history, hitting, pitching, and records

## Deployment

Hosted via GitHub Pages at [mshipley9.github.io/linnmar-baseball](https://mshipley9.github.io/linnmar-baseball)

## Notes

- All data is embedded directly in each HTML file — no server or database required
- To update data, re-generate the file with the updated spreadsheet
